# AI 강사 Agent v2.0

> PPT 파일을 입력하면 슬라이드 내용을 분석하고, 페이지별 강의 내용을 생성한 뒤,  
> 스크립트·음성·영상을 자동으로 만들어 최종 강의 영상으로 합쳐주는 **LangGraph 기반 AI 강사 에이전트**입니다.

---

## 프로젝트 개요

`AI 강사 Agent v2.0`은 발표 자료(PPT/PPTX)를 입력으로 받아,  
각 슬라이드의 텍스트·표·이미지를 분석하고, 부족한 설명은 외부 검색으로 보완한 뒤,  
강의 스크립트 생성 → TTS 음성 변환 → 슬라이드 영상 합성 → 전체 영상 병합까지 수행하는 프로젝트입니다.

최종적으로는 사용자가 업로드한 PPT를 바탕으로 **자동 강의 영상**을 생성하며,  
노트북 기반 실행뿐 아니라 **Gradio Web UI**를 통해서도 사용할 수 있도록 구성되어 있습니다.

---

## 핵심 기능

- **PPT 입력**
  - 사용자가 업로드한 PPTX 파일을 입력으로 사용

- **슬라이드 정보 분해**
  - 슬라이드별 텍스트, 표, 이미지, 제목, 스냅샷 추출
  - 불필요한 정보를 정리하고 후속 단계에서 활용할 수 있는 구조화된 state 생성

- **외부 검색 보강**
  - 슬라이드 제목을 기준으로 SerpAPI(Google 검색) 결과를 수집
  - 부족한 배경지식이나 부연 설명에 활용

- **페이지별 강의 내용 생성**
  - 각 슬라이드의 핵심 내용을 요약·정리
  - 단순 PPT 낭독이 아니라 설명형 콘텐츠로 확장

- **강의 스크립트 생성**
  - 전체 강의 흐름을 고려하여 페이지별 발표 스크립트 작성
  - 앞뒤 슬라이드 맥락을 함께 반영하는 구조

- **TTS 음성 변환**
  - OpenAI TTS 모델을 이용해 강의 스크립트를 음성(mp3)으로 변환
  - tone / voice 설정 가능

- **영상 제작**
  - 슬라이드 스냅샷 이미지 + 오디오를 합성해 페이지별 mp4 생성
  - 이후 전체 페이지 영상을 이어 붙여 최종 강의 영상 생성

- **LangGraph 기반 Agent 워크플로우**
  - 전체 파이프라인을 노드 단위로 구성하고 그래프로 연결

- **Gradio Web UI**
  - PPT 업로드, 강의 톤 입력, Voice 선택
  - 최종 영상 미리보기 및 다운로드 지원

---

## 전체 파이프라인

1. **PPT 업로드**
2. **슬라이드 파싱**
   - 텍스트 / 표 / 이미지 / 제목 / 스냅샷 추출
3. **외부 검색**
   - 슬라이드 제목 기반 웹 검색 요약
4. **페이지별 내용 생성**
5. **강의 스크립트 생성**
6. **TTS 음성 생성**
7. **슬라이드별 영상 생성**
8. **전체 영상 병합**
9. **Web UI에서 재생 / 다운로드**

## 주요 구성 요소

### 1. 슬라이드 파싱
PPT를 불러와 슬라이드별로 다음 정보를 추출합니다.

- 제목
- 본문 텍스트
- 표 데이터
- 이미지
- 스냅샷 이미지

이 과정에서 각 슬라이드는 state 내부의 구조화된 데이터로 저장됩니다.

### 2. 외부 검색 노드
슬라이드 제목을 기준으로 SerpAPI를 호출해  
관련 검색 결과(title, snippet, link)를 요약 문자열로 정리합니다.

이를 통해 슬라이드 원본 내용이 부족한 경우  
강의 생성 단계에서 설명을 보강할 수 있습니다.

### 3. 페이지 내용 생성
슬라이드 텍스트, 표, 이미지, 외부 검색 결과를 함께 참고하여  
슬라이드별 설명 콘텐츠를 생성합니다.

### 4. 스크립트 생성
현재 슬라이드만 보지 않고,  
전체 강의 흐름을 고려해 보다 자연스러운 발표 스크립트를 생성합니다.

### 5. TTS 생성
생성된 스크립트를 기반으로 mp3 파일을 만듭니다.  
voice와 tone을 조절할 수 있어 강의 스타일을 바꿀 수 있습니다.

### 6. 영상 생성 및 병합
각 슬라이드의 스냅샷 이미지를 배경으로,  
해당 슬라이드의 오디오를 합성해 mp4를 생성합니다.  
이후 모든 페이지 영상을 순서대로 합쳐 최종 강의 영상을 만듭니다.

### 7. Gradio UI
사용자는 Web UI에서 다음을 입력/선택할 수 있습니다.

- PPTX 업로드
- 강의 톤 입력
- TTS Voice 선택

출력:
- 최종 강의 영상 미리보기
- 최종 영상 다운로드

## LangGraph 노드 구성

프로젝트는 다음과 같은 노드 단위 구조로 설계되어 있습니다.

- `parse_all`
- `tool_search`
- `gen_page`
- `gen_script_ctx`
- `tts`
- `make_video`
- `accumulate_and_step`
- `concat`

이 흐름을 통해 PPT 분석부터 최종 영상 생성까지 자동화합니다.

실행 후 생성되는 주요 결과물:

- 페이지별 오디오 (`mp3`)

## 실행 방법

### 1. Google Colab에서 노트북 실행
노트북을 Colab에 업로드한 뒤 순서대로 실행합니다.

### 2. API Key 준비
`api_key.txt` 또는 환경변수에 아래 키를 준비합니다.

```txt
OPENAI_API_KEY=...
SERPAPI_API_KEY=...
```

### 3. PPTX 업로드
노트북 또는 Gradio UI에서 강의용 PPTX 파일을 업로드합니다.

### 4. 파이프라인 실행
LangGraph 앱을 실행하면 전체 파이프라인이 순차적으로 수행됩니다.

### 5. 결과 확인
- 콘솔에서 슬라이드별 처리 결과 확인
- 노트북에서 영상 미리보기 확인
- Gradio에서 다운로드 가능

## 프로젝트 의의

이 프로젝트는 단순한 PPT 요약 도구가 아니라,  
**문서 분석 → 정보 보강 → 스크립트 생성 → 음성 합성 → 영상 제작**까지 이어지는  
멀티모달 교육 콘텐츠 생성 파이프라인을 구현한 프로젝트입니다.

특히 LangGraph를 기반으로 전체 과정을 노드 단위로 분리해 자동화했다는 점에서,  
AI Agent 설계와 실제 서비스형 워크플로우 구현 경험을 보여주는 프로젝트라고 볼 수 있습니다.
